GTM tracking reference

Both forms submit via GET to thank-you.html, appending hidden fields as URL parameters (form_id, form_name, event). The thank-you page reads those params and pushes them to window.dataLayer, making them available to GTM triggers. The page <title> is also set dynamically per form, so you can use GTM's Page Title variable as a secondary differentiator.
